# Developer Tools

Developer environment tooling for Claude Code.

## Skills

### devcontainer-generator

Generate production-ready devcontainer configurations by analyzing repository tech stack. Supports Node.js, Python, .NET, Go, Rust, and Java with optional services (PostgreSQL, Redis, MongoDB, etc.).

**Usage:**

```
/developer-tools:devcontainer-generator
```

## Documentation

See `docs/devcontainer-generator/` for detailed documentation on stacks, services, and customization.
