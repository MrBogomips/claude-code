- **A** — Accountable: autorità decisionale finale

## Legal Boilerplate

### Clausola di Riservatezza (standard)
Tutte le informazioni scambiate tra le parti in relazione al presente documento saranno trattate come riservate e non saranno divulgate a terzi senza previo consenso scritto, salvo quanto previsto dalla legge.

### Clausola di Proprietà Intellettuale (work-for-hire, standard)
Tutti i deliverable prodotti nell'ambito del presente SOW sono da considerarsi opera su commissione. Al completamento del pagamento, tutti i diritti di proprietà intellettuale sono trasferiti al Committente, ad eccezione della proprietà intellettuale preesistente e dei componenti di terze parti che rimangono sotto licenza originale.

### Clausola di Garanzia (standard)
Il Fornitore garantisce che tutti i deliverable saranno conformi ai criteri di accettazione specificati nel presente documento per un periodo di [DURATA] dalla data di accettazione finale. Durante il periodo di garanzia, il Fornitore correggerà eventuali difetti senza costi aggiuntivi.

|---------|---------|---------|
| Capitolato Tecnico | Technical Specifications | Formal technical requirements document in public tenders |
| Disciplinare di Gara | Tender Rules | Rules governing the tender process |
| Bando di Gara | Call for Tenders | Public announcement of the tender |
| Offerta Tecnica | Technical Proposal | Vendor's technical response to the tender |
| Offerta Economica | Financial Proposal | Vendor's pricing response |
| Stazione Appaltante | Contracting Authority | The public entity issuing the tender |
| Operatore Economico | Economic Operator | The vendor/bidder |
| RUP | Unique Process Manager | Responsible official for the procurement |
| CIG | Tender Identification Code | Unique code for each tender (Codice Identificativo Gara) |
| CUP | Unique Project Code | Unique code for public investment projects |
| DURC | Regularity Certificate | Certificate of compliance with social contributions |
| Codice degli Appalti | Public Contracts Code | D.Lgs. 36/2023 — primary legislation governing public procurement |
