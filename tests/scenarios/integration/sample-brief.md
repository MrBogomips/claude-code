# Project Brief: HomeAI Property Management Platform

## Parties
- **Client**: Meridian Properties Group S.r.l. (Milan, Italy)
- **Vendor**: TechForge Solutions Ltd (London, UK)

## Background
Meridian Properties manages 2,500+ residential and commercial properties across Northern Italy. Their current property management relies on spreadsheets, email, and a legacy desktop application (PropManager 2.0, circa 2015). They need a modern cloud platform to streamline operations, improve tenant satisfaction, and enable data-driven decision-making.

## High-Level Scope
Build a multi-tenant SaaS platform ("HomeAI") that provides:
- Property portfolio management (listings, units, tenants, leases)
- AI-powered maintenance request triage and routing
- Financial management (rent collection, expense tracking, reporting)
- Tenant portal (self-service, communication, document access)
- Owner/investor dashboard (portfolio performance, ROI analytics)

## Phases
1. **Discovery & Foundation** (estimated 8 weeks): requirements validation, architecture design, core infrastructure setup, CI/CD pipeline
2. **Core Platform** (estimated 16 weeks): property management module, tenant management, lease lifecycle, basic financial module
3. **AI & Advanced Features** (estimated 12 weeks): AI maintenance triage, owner dashboard, advanced analytics, mobile app

## Team Hints
- Project requires 4-6 developers, 1 PM, 1 UX designer, 1 QA lead
- Client will provide 2 SMEs (operations manager + finance lead) at 20% allocation
- Need Italian-speaking team members for tenant-facing features

## Budget Range
EUR 450,000 - 600,000 for the full project (CAPEX)
Ongoing OPEX estimated at EUR 8,000-12,000/month post-launch

## Timeline
- Project start: Q2 2026
- Phase 1 completion: end of Q2 2026
- Phase 2 completion: end of Q3 2026
- Phase 3 completion: Q1 2027
- Go-live: Q1 2027

## Key Concerns
- Data migration from PropManager 2.0 (uncertain data quality)
- GDPR compliance for tenant PII
- Integration with existing accounting system (SAP Business One)
- Multi-language support (Italian primary, English secondary)
